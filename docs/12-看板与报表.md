# 看板与报表

## 一、看板设计

### 看板类型

按用户角色划分：

- **管理面板**：PL、项目经理、技术经理
- **项目成员面板**：职能部门leader、工程师、技师

### 面板区域

1. **链接项区域**：快速跳转+数量展示
2. **报表区域**：选择预置报表展示图形

### 链接项定义

| 链接项 | 跳转目标 | 数量计算规则 |
|--------|---------|-------------|
| 我的项目 | 项目查询界面 | 当前用户参与的项目中，除已关闭和取消的项目数量 |
| 工作流待办 | 审批查询界面 | 当前用户待审批的事项数 |
| 风险/问题待办 | 风险/问题管理界面 | 进行中的风险/问题数量 |
| 行动项 | 行动项界面 | 跟进中的行动项数量 |

### 报表区域

- 可从预置报表中选择展示
- 选择后展现图形报表
- 针对单个用户的选择进行保存
- 可进行参数选择（如项目号），最多3个参数
- 实际总金额需在看板中体现
