# 09-项目验收模块

## 一、验收流程

1. 项目竣工后PM创建验收单
2. PM组织线下评审
3. 满足验收标准 → PM在HPM确认 → 提交审批
4. 项目负责人审批 → 完成

---

## 二、验收单信息

### 2.1 基本信息

| 字段 | 说明 |
|------|------|
| 验收单号 | 系统自动生成，唯一标识 |
| 项目编号 | 关联的项目编号 |
| 项目名称 | 关联的项目名称 |
| 验收状态 | 草稿 / 进行中 / 驳回 / 完成 |
| 创建人 | 验收单创建人（PM） |
| 创建时间 | 验收单创建时间 |
| 确认时间 | PM确认时间 |
| 完成时间 | 审批完成时间 |

### 2.2 编号规则

**验收单号格式**
```
Y + 年份(4位) + 项目号
```

**示例**：Y2019PRJ001
- Y：验收单标识
- 2019：年份
- PRJ001：项目号

### 2.3 审批流程

| 项目类型 | 审批流程 |
|----------|----------|
| 大项目 | PM审批 → 敖经理直属领导审批 → 完成 |
| 小项目 | PM审批 → 舒经理直属领导审批 → 完成 |

**说明**：
- 第一级：PM审批（确认验收内容）
- 第二级：项目负责人审批（最终批准）

---

## 三、功能点清单

### 3.1 新建验收单
- 创建验收单
- 单号自动生成
- 状态默认为"草稿"

### 3.2 确认验收单
- PM确认验收内容
- 状态变为"进行中"
- 记录确认时间

### 3.3 保存验收单
- 保存验收信息
- 支持多次保存
- 草稿和进行中状态均可保存

### 3.4 删除验收单
- 仅草稿状态可删除
- 删除后不可恢复

### 3.5 提交审批
- 推送给直属领导
- 触发审批流程
- 校验必填项

### 3.6 VPP核对
- 按钮跳转到VPP界面确认
- 小项目没有VPP核对环节

### 3.7 任务验收
- 自动带出勾选"是否交付物"的任务
- 展示字段：任务名称、交付物、交付物名称
- 校验交付物完成情况

### 3.8 附件管理
- 维护验收相关文件路径和名称
- 支持多附件上传
- 提交审批时校验必需文件

## 五、页面要求

### 5.1 验收单列表页面
- 查询条件：验收单号、验收状态、项目名称、项目编码
- 列表字段：验收单号、项目编号、项目名称、验收状态、创建时间
- 操作按钮：查看、编辑、删除（草稿状态）

### 5.2 验收单创建/编辑页面

#### 基本信息区域
- 项目选择：下拉选择（已竣工项目）
- 验收单号：系统自动生成，只读
- 验收状态：当前状态，只读

#### 验收内容区域
- 验收标准：文本域，描述验收标准
- 验收结论：通过/不通过
- 验收备注：补充说明

#### 任务验收区域
- 表格展示：自动带出带交付物的任务
- 字段：任务名称、交付物状态、交付物名称、是否完成
- 校验：所有任务交付物必须完成

#### 附件区域
- 文件上传：支持多文件
- 路径输入：外部系统文件路径
- 附件列表：已上传附件展示

#### 操作按钮区域
- 保存：保存当前内容
- 确认：确认验收内容（草稿→进行中）
- 提交审批：触发审批流程
- VPP核对：跳转到VPP系统（大项目）
- 删除：删除验收单（仅草稿）

### 5.3 验收单审批页面
- 验收详情展示
- 审批意见输入框
- 批准/驳回按钮
- 附件查看

### 5.4 验收单详情页面
- 只读展示验收信息
- 审批流程跟踪
- 任务验收情况
- 附件列表

## 六、校验规则

| 序号 | 校验项 | 规则说明 |
|------|--------|----------|
| 1 | 项目状态 | 只有竣工项目可创建验收单 |
| 2 | 验收单唯一性 | 一个项目只能有一个进行中的验收单 |
| 3 | 删除限制 | 仅草稿状态可删除 |
| 4 | 提交审批校验 | 必填项已填写，任务交付物已完成 |
| 5 | 附件校验 | 提交审批时校验所有必需文件均已上传 |
| 6 | VPP核对 | 大项目必须通过VPP核对才能提交审批 |
| 7 | 状态流转 | 草稿→进行中→完成/驳回 |

---

## 七、计算公式

### 7.1 验收单号生成
```
验收单号 = "Y" + 年份(4位) + 项目号
```

**示例计算**
- 项目号：PRJ001
- 当前年份：2019
- 验收单号：Y2019PRJ001

### 7.2 任务完成率
```
任务完成率 = 已完成任务数 / 总验收任务数 × 100%
```

### 7.3 交付物完成率
```
交付物完成率 = 已完成交付物数 / 总交付物数 × 100%
```

## 八、前置/后置条件

### 8.1 前置条件

| 操作 | 前置条件 |
|------|----------|
| 创建验收单 | 用户为PM或管理员，项目状态为"竣工" |
| 编辑验收单 | 验收单状态为"草稿"或"进行中"，用户有权限 |
| 确认验收单 | 验收单状态为"草稿"，用户为PM |
| 删除验收单 | 验收单状态为"草稿"，用户为创建人或管理员 |
| 提交审批 | 验收单状态为"进行中"，必填项完整，大项目已通过VPP核对 |
| 审批验收 | 验收单状态为"进行中"，用户为项目负责人 |

### 8.2 后置条件

| 操作 | 后置条件 |
|------|----------|
| 创建验收单 | 状态为"草稿"，记录创建信息 |
| 确认验收单 | 状态变为"进行中"，记录确认时间 |
| 提交审批 | 通知审批人，进入审批流程 |
| 审批通过 | 状态变为"完成"，项目状态更新，通知相关人员 |
| 审批驳回 | 状态变为"驳回"，通知PM，可重新编辑提交 |
| 删除验收单 | 删除记录，释放项目验收状态 |

## 九、任务验收详细规则

### 9.1 自动带出规则
- 自动带出项目中勾选"是否交付物"的任务
- 展示字段：任务名称、交付物、交付物名称

### 9.2 任务验收状态

| 状态 | 说明 |
|------|------|
| 待验收 | 任务已完成但未验收 |
| 已验收 | 任务已通过验收 |
| 不通过 | 任务未通过验收 |

### 9.3 验收校验
- 提交审批时校验所有任务交付物状态为"已验收"
- 如有未完成任务，提示并阻止提交

## 十、附件管理详细规则

### 10.1 附件类型

| 类型 | 说明 | 是否必需 |
|------|------|----------|
| 验收报告 | 项目验收报告文档 | 是 |
| 测试报告 | 项目测试报告 | 是 |
| 用户手册 | 项目交付的用户手册 | 视项目类型 |
| 其他文档 | 其他相关文档 | 否 |

### 10.2 附件上传方式

| 方式 | 说明 |
|------|------|
| 本地上传 | 从本地选择文件上传 |
| 路径填写 | 填写外部系统文件路径 |

### 10.3 附件校验规则
- 提交审批时校验必需附件是否已上传
- 文件格式校验
- 文件大小限制
